  TASK-2: Loan Approval Prediction
Objective

Build a supervised machine learning model to predict loan approval status based on borrower characteristics. The project emphasizes data preprocessing, class imbalance handling, model comparison, and business interpretation.

Key Tasks
Data cleaning and preprocessing
Handling missing values
Encoding categorical variables
Feature scaling
Addressing class imbalance using techniques such as:
SMOTE
Undersampling
Class Weights
Training and evaluating multiple machine learning models
Comparing model performance using:
Precision
Recall
F1-Score
ROC-AUC
Models Evaluated
Logistic Regression
Decision Tree
Random Forest
Other tree-based models (if applicable)
Results

The best-performing model was selected based on evaluation metrics and its ability to correctly identify approved and rejected loan applications.

Business Insights
Identified the most influential borrower characteristics affecting loan approval.
Demonstrated how predictive models can help financial institutions reduce risk and improve decision-making.
Provided recommendations for optimizing loan approval processes.
